description: libmodulor is an opinionated TypeScript library to create business oriented applications.
---

Applications created with `libmodulor` have **6 main properties**

<Cards>
    <Card>Strictly typed with explicit business data types</Card>
    <Card>Fully typed e2e without code generation</Card>
    <Card>Auto documented</Card>
    <Card>Auto tested</Card>
    <Card>Multi platforms/runtimes</Card>
    <Card>Runnable anywhere</Card>
</Cards>
